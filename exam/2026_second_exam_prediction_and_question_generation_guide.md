## 4.2 예상 낚시

### 중첩 리스트

```python
a = [[1], [2], [3]]
```

`a[1]`은 `2`가 아니라 `[2]`다.

### 리스트 `+=`

```python
a[1] += a[0]
```

각 값이 리스트라면 산술 덧셈이 아니라 리스트 확장이다.

### 슬라이싱

```python
a[::-1]
a[::-2]
a[::2]
a[1::2]
a[1:4]
```

끝 인덱스는 포함되지 않는다.

### 얕은 복사

```python
b = a[:]
b = a.copy()
```

중첩 리스트의 내부 객체는 공유될 수 있다.

### 딕셔너리

```python
d[key] = d.get(key, 0) + 1
```

같은 키를 다시 대입하면 기존 값이 덮어써진다.

### 문자열 처리

```python
split()
join()
replace()
strip()
count()
```

`join`은 구분자 문자열이 호출한다.

```python

```

### `range`

```python
range(1, 4)
```

결과는 `1, 2, 3`이다.

# 5. SQL 및 데이터베이스 예상 출제 유형

## 5.1 SQL

### `DISTINCT`

```sql
SELECT DISTINCT a, b
FROM table_name;
```

`a`, `b` 각각이 아니라 `(a, b)` 조합의 중복을 제거한다.

### `COUNT`

```text
COUNT(*) → 전체 행
COUNT(column) → NULL 제외
COUNT(DISTINCT column) → NULL 제외 + 중복 제거
```

### `GROUP BY`와 `HAVING`

```text
WHERE → 그룹화 전 개별 행 조건
HAVING → 그룹화 후 그룹·집계 조건
```

### JOIN

- `INNER JOIN`
- `LEFT OUTER JOIN`
- 중복 키로 인한 행 증가
- NULL 포함 여부
- 조인 결과 행 수 계산

### 서브쿼리

```sql
WHERE score > (
    SELECT AVG(score)
    FROM student
)
```

`>`와 `>=`, 전체 평균과 그룹 평균을 구분한다.

## 5.2 데이터베이스 이론

- 1NF, 2NF, 3NF, BCNF
- 부분 함수 종속
- 이행 함수 종속
- 결정자
- 후보키, 기본키, 대체키, 슈퍼키, 외래키
- 개체·참조·도메인 무결성
- 트랜잭션 ACID
- Commit, Rollback
